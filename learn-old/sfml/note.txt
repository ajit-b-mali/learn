<---- Other     ---->

- SFML -- Simple and Fast Multimedia Library
- Five modules
	1. System -- Core module, provide 2D and 3D vector classes, clocks, threads, unicode strings, etc
	2. Window -- Create application windows, collect user inputs
	3. Graphics -- 2D rendering, images, texts, shapes, colors
	4. Audio -- For sound, load music
	5. Network -- send data over LAN or the Internet, work with protocols (like HTTP,FTP)
- Learn Vector alegebra for games
- Time handling and Frame updates
	- Frame Independent Movement
	- Fixed Time Steps
	- Interpolations between two states to smoothen the game flow -> http://gafferongames.com/game-physics/fix-your-timestep
	- Vertical synchronization
- File path are related to working direectories
- Real time rendering
- Render-on=demand
- Double Buffering
- Shaders -- sfml suppports only vertex and fragment
	bloom shader -> amplifies bright parts of the scene
	toon shader -> makes objects look like in a cartoon
	blur effect -> simulates flickering hot air
- Acquiring, releasing, and accessing resources
