# Introduction to Node.js

- built on JavaScript
- a JavaScript runtime
- allows to run JavaScript on server
- uses V8 -- JS engine/compiler built by Google using C++ to run JS on browser
- add some extra features to JS -- like working with local file system, 
- can not access DOM -- Document Object Model
- `node --version` -> v18.19.1 lts (using), latest -> v20.17.0
- `npm --version` -> 10.8.2
- official [site](www.nodejs.org)

# Running Node.js

## Using REPL

- `node` command is used to enter in REPL mode
- `.exit` / `<Ctrl> + d` / `<Ctrl> + c` * 2 -> to exit REPL mode

## Using `.js` file

- `.js` extension used for JS files
- `node ---.js` to execute that `.js` file
- see [example](/projects/hello/hello.js)

## Output to file

- `const fs = require('fs')` -- importing fs module
- `fs.writeFileSync("path/to/file.txt", "Content of file")` -- write a file to hard drive
