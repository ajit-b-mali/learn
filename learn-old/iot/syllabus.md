**NOTE - for MCA**

# Course Prerequisite

- Basic of computer network

# Course Objectives

- To know the IoT fundamentals and understand the technologies
- To learn the basics of Arduino and Raspberry Pi installation, setup and configuration
- To learn IoT protocols, cloud platforms and security issues in IoT

# Course Outcomes

1. Explain fundamentals, architecture and framework of IoT
2. Analyze IoT protocols for making IoT devices communication
3. Explain python programming for development of IoT application
4. Design cloud based IoT system
5. Develop secured IoT application
6. Cunstruct real time IoT application
